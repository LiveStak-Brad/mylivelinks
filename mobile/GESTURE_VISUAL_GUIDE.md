## Accessibility Notes

```
Long-Press:
    Duration: 450ms (standard)
    Visual feedback: Immediate (scale starts)
    Audio feedback: None (future: haptic)
    
Double-Tap:
    Max interval: 300ms between taps
    Visual feedback: Immediate (expand)
    Audio feedback: None (future: haptic)
    
Buttons:
    Size: Meets minimum 44x44 dp
    Labels: "Done", "Close" (for screen readers)
    High contrast: White on blue/black
```

## Performance Characteristics

```
Gesture Detection:
    Long-press: UI thread (Reanimated)
    Double-tap: UI thread (Reanimated)
    Swipe: UI thread (Reanimated)
    
State Updates:
    Edit mode: 1 state update
    Focus mode: 1 state update
    Minimal re-renders
    
Animations:
    All on UI thread (60fps)
    No JS bridge crossing
    Smooth transitions
```
