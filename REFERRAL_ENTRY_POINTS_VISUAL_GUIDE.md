### Key Visual Elements (ReferralCard)

1. **Background Gradient**
   - `from-purple-600 via-pink-600 to-orange-500`
   - Decorative blur circles for depth
   - Shadow: `shadow-2xl`

2. **Icon Badge**
   - 56px Ã— 56px rounded square
   - White/20 background with backdrop blur
   - Users icon (28px)

3. **Typography**
   - Title: 2xl/3xl, bold, white
   - Body: base/lg, white/90
   - Hint: sm, white/80 with sparkles emoji

4. **Button States**
   - Default: White bg, purple text
   - Hover: Gray-100 bg, scale-105 transform
   - Active: Shadow-xl

5. **Spacing**
   - Outer padding: 6-8 units (24-32px)
   - Section spacing: 4-6 units (16-24px)
   - Border radius: 2xl (16px)

### Key Visual Elements (ReferralProgressModule)

1. **Header Section**
   - Icon badge with accent color tint (20% opacity)
   - Title + subtitle layout
   - Rank badge on right (accent color bg)

2. **Stats Cards**
   - Left: Blue gradient (Joined count)
   - Right: Green gradient (Active count)
   - Large number (3xl, bold)
   - Icon + label at top
   - Descriptive text at bottom

3. **Progress Bar**
   - Label with percentage on right
   - Gray background track
   - Accent color fill (animated width)
   - Helper text below

4. **Rank Info Box**
   - Accent color background (8% opacity)
   - Award icon badge
   - Rank number + context text
   - Trending up icon

5. **Action Buttons**
   - Primary: Accent color background
   - Secondary: Outline style
   - Flex layout (stacks on mobile)
   - Icons for clarity
